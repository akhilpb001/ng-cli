# Angular CLI Docker Images
Dockerfiles for Angular CLI images.

## Docker Commands
```
docker pull akhilpb001/ng-cli:<version>
```
```
docker run -it --rm -p 4200:4200 -v "$PWD":/app akhilpb001/ng-cli:<version> sh
```

## Available Versions
- 11.2.18 (v11-lts)
- 11.2.14
- 11.2.8
- 10.2.3
- 9.1.15 (v9-lts)
